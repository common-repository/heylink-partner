The official Heylink order tracking plugin for WooCommerce.

== Description ==

The Heylink Partner plugin will automatically track <a href="https://wordpress.org/plugins/woocommerce/" target="_blank">WooCommerce</a> orders made from your affiliate links.<br />
Joining our affiliate tracking platform was never easier.
Visit our website to learn more <a href="https://heylink.com/" target="_blank">heylink.com</a>

== Installation ==

1. Visit <strong>Plugins > Add New</strong>
2. Search for "<strong>Heylink Partner</strong>"
3. Download and Activate the plugin.

== Frequently Asked Questions ==

= What are the requirements? =

The minimum required software versions are:
* PHP 5.6
* WordPress 5
* WooCommerce 4

= Which orders are tracked? =

By default, the plugin will track all orders.
